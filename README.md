# Order Processing System

A production-grade backend system built using ASP.NET Core, showcasing
event-driven architecture, CQRS, async processing, observability, and
cloud-native practices.

## Tech Stack
- ASP.NET Core (.NET 8)
- RabbitMQ (Event-driven messaging)
- PostgreSQL (Write DB)
- Redis (Caching, Idempotency)
- CQRS + MediatR
- Docker & Docker Compose
- Nginx (Load Balancer)
- OpenTelemetry + Grafana (Observability)
- Rate Limiting, Retry with Backoff
- Azure-ready architecture
